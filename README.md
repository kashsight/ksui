## 📸 See it in Action

`cvui` provides a polished, icon-rich terminal experience directly in Termux:

### 📁 Iconified Data Presentation (`ls`)
Never guess the file type again. Built-in `ls`, `ll`, and `la` aliasing replaces standard output with visual cues:

```text
📂  src/          📝  README.md      ⚙️  main.py
🏗  install.sh    📄  notes.txt      ⚙️  script.go
```

### 🧠 AI-Powered Interaction
Launch `cvui` and ask anything immediately using the `ask` command:

```bash
> ask What is the capital of Kenya?
🤖 KAI: The capital of Kenya is Nairobi.
```

---

## ✨ Features

- 🔐 **Authentication**: First-run account setup with local security.
- 🔄 **Auto-Updates**: Checks for updates on launch; prompts to pull changes automatically.
- 🗣 **KAI Voice**: Live TTS for greetings, jokes, facts, and creative timer alerts.
- 🔍 **History Search**: Custom in-house prefix search via Up/Down arrows.
- ⚡ **Zero Bloat**: Streamlined, efficient shell experience.

## 🚀 Installation

```bash
pkg update && pkg upgrade -y && pkg install git -y && git clone https://github.com/cybervaultke/cvui-legacy.git && cd cvui-legacy/install && bash install.sh
```
